### Simple wrapper for sge qsub which provides the functionality of the "-sync" option

*python requires: >=3.5*

#### Install

> git repo (for recommend)

```
pip install git+https://github.com/yodeng/qsubwt.git
```

> Pypi

```
pip install qsubwt -U
```



#### Help

